# search_stalls

## 功能

帮助用户查找摊位，可按关键词、分类、商品、位置进行筛选。

## 触发语

- "附近有什么摊位"
- "找糖画摊"
- "有没有农家特产"
- "推荐评分高的摊位"

## 参数

```json
{
  "keyword": "糖画",
  "category": "非遗好物"
}
```

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `keyword` | string | 条件必填 | 用户搜索词，可包含摊位、商品、位置 |
| `category` | string | 条件必填 | 分类，如 `地方特色`、`农家特产`、`非遗好物` |

`keyword` 和 `category` 至少需要一个。“附近摊位”“帮我找摊位”这类泛化表达不算有效参数，必须先追问。

有效参数判断：

- `category` 只能是系统分类，如 `地方特色`、`农家特产`、`非遗好物`。
- `keyword` 可以是用户明确说出的商品、摊位、摊主、位置或特征。
- “附近”“推荐”“好摊位”“吃的”不是足够具体的搜索参数。
- 模型不能把“附近摊位”自动改成某个分类。

## 参数不足响应

```json
{
  "type": "ask_clarification",
  "label": "补充信息",
  "route": "",
  "payload": {}
}
```

追问文案：

`你想找哪类摊位？可以说地方特色、农家特产、非遗好物，或告诉我具体商品/位置。`

## 响应 action

```json
{
  "type": "open_route",
  "label": "查看摊位列表",
  "route": "/stalls",
  "payload": {
    "keyword": "糖画",
    "category": "非遗好物"
  }
}
```

## 前端行为

展示摊位卡片，用户确认后跳转 `/stalls`。
