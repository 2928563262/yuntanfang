# 云摊坊 MVP 待开发清单

本文档只保留适合 1-2 天内完成并提交 MVP 版本的核心开发内容。已完成内容见 `docs/development-progress.md`。

## MVP 目标

- 普通用户、商家、管理后台三端都能登录并完成核心业务。
- 用户能找摊位、看商品、下单、查订单、评价、投诉、看消息。
- 商家能管理商品、处理订单、回复评价、查看入驻/资质/摊位状态。
- 管理后台能做四级审核、处理投诉、查看基础数据。
- Agent 能作为“快速开始”入口，辅助用户找摊位、下单、评价、投诉或定位功能入口。
- 一条自动化脚本可以验证主链路没有断。
